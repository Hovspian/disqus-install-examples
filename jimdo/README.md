# Jimdo installation instructions
 
1. Create your shortname at the [registration page](https://disqus.com/admin/create/).
2. On the next page, select the **Universal Embed Code** option.

  **The following instructions will need to be completed for each post that you would like Disqus to appear on.**

3. Click on the **Blog** icon in the Site Admin. Then click on the blog post that you want to change, or create a new post. 
4. Click on the blog post's **Title** to view the settings, then click on the speech bubble icon to turn off Jimdo's comments. 
5. In the **Additional Elements...** section of the **Elements** menu, select **[Widget/HTML Element](http://support.jimdo.com/basics/working-with-elements/widget-html/)** and paste [Universal Embed Code](https://disqus.com/admin/install/platforms/universalcode/)
 
[Continue to the getting started guide](https://help.disqus.com/customer/portal/articles/1264625-getting-started).
